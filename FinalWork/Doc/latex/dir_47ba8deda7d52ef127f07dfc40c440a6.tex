Программа Temperature Statistics предназначена для работы с данными о температуре. Она позволяет\+:


\begin{DoxyEnumerate}
\item Хранить записи о температуре с указанием даты и времени
\item Анализировать температурные данные
\item Отображать статистику по месяцам и годам
\item Загружать данные из CSV-\/файлов
\end{DoxyEnumerate}\hypertarget{ReadMe.md_autotoc_md2}{}\doxysubsection{\texorpdfstring{Функциональность}{Функциональность}}\label{ReadMe.md_autotoc_md2}

\item Добавление новых записей о температуре
\item Удаление существующих записей
\item Просмотр статистики\+:

3.\+1. Средняя, минимальная и максимальная температура по месяцам

3.\+2. Средняя, минимальная и максимальная температура по годам
